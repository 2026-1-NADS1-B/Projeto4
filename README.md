## Descrição

<p align="center">
<img width="1024" height="1024" alt="Logo do Sistema Messier" src="https://github.com/user-attachments/assets/44cd3d89-9161-4016-9ce3-ab0f9c371bd1" />
</p>

O Sistema Messier é um aplicativo desktop desenvolvido para gerenciar assinaturas de jogos educacionais voltados para escolas de ensino fundamental e médio. A plataforma permite o controle de pacotes contratados, limite de acessos mensais e validação de usuários com base em endereços IP autorizados.

O sistema realiza a verificação de acesso utilizando regras de negócio que envolvem pacote ativo, autorização de IP e limite de uso, além de registrar todas as interações em logs. O projeto integra conceitos de banco de dados, redes de computadores, lógica de programação e modelagem de sistemas, simulando uma aplicação real.
